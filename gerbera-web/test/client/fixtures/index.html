Set up Gerbera with persistent content

A bit more complicated but safer in the long run
  • Add autoscan entry in config.xml (See documentation)
  • Adjust further settings as required
  • Restart Gerbera service
  • Wait for import to complete
  • Check logs for error messages

Set up Gerbera with dynamic content

Easier to create but all setup is lost when your database is deleted or corrupted
  • Go to File System view
  • Look for folders containing your media files
  • Click Add Autoscan Item
  • Wait for import to complete
  • Check logs for error messages
